
# Template From WebWind 🌪

## Local Dev
1. install node, get install pkg from https://nodejs.org/en/download
2. run `npm install`
3. run `npm run dev`

## Remote Development
1. run `npm run build`, and get `dist` folder
2. put the `dist` folder to https://app.netlify.com/
